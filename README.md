# Gemini Project

이 프로젝트는 Gemini CLI를 사용하여 생성 및 관리되는 작업 공간입니다.

## 프로젝트 구성

### 1. Todo App (`todo-app/`)
React, TypeScript, Vite, Tailwind CSS로 구축된 간단한 Todo 관리 애플리케이션입니다.
- **실행 방법:**
  ```bash
  cd todo-app
  npm install
  npm run dev
  ```

### 2. Python 유틸리티 (`sum.py`)
사용자로부터 두 개의 숫자를 입력받아 합계를 출력하는 간단한 파이썬 스크립트입니다.
- **실행 방법:**
  ```bash
  python sum.py
  ```

### 3. 문서
- **GEMINI.md**: Gemini CLI 컨텍스트 메모리 파일

## 환경 설정
- `.gitignore`: 버전 관리에서 제외할 파일 목록 설정 (예: `.gemini/`)
